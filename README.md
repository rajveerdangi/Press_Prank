## How to Install the Chrome Extension

### 1. Download the Extension

* Download the file: **Press_Extension.zip**

---

### 2. Extract the Files

* Unzip the downloaded file
* Make sure the folder contains files like `manifest.json`
* ⚠️⚠️ Do NOT select the zip file during installation—use the extracted folder ⚠️⚠️

---

### 3. Open Chrome Extensions Page

* Open Google Chrome
* Click the **three dots (⋮)** in the top-right corner
* Go to: **Extensions → Manage Extensions**

---

### 4. Enable Developer Mode

* Turn ON **Developer mode** (top-right corner)

---

### 5. Load the Extension

* Click **“Load unpacked”**
* Select the **extracted extension folder**

---

### 6. Pin the Extension (Optional but Recommended)

* Click the **Extensions icon**
* Find your extension
* Click the **pin icon** to keep it visible

---

## You're All Set! ✅✅

* The extension is now installed
* Try clicking or typing to test the sounds
* Or… install it on a friend’s laptop for maximum fun 😏

---

## ⚠️ Important Tips  ⚠️ 

* Always select the **unzipped folder**, not the `.zip` file
* Ensure `manifest.json` is inside the selected folder
* If something doesn’t work, click **Reload** on the Extensions page
